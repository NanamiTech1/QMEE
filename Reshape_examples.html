Look at some example data that comes with the package:

smiths

The default gather() operation squashes everything too far, including the subject name and time in the value column …

gather(smiths)
## Error in is.data.frame(x): object 'smiths' not found

We can specify that we only want to gather the age and weight variables (however, we have to specify the name of key and value columns explicitly).

print(smelt <- gather(smiths, key="var", value="value",
       c(age,weight)))
## Error in is.data.frame(x): object 'smiths' not found

Alternatively we could specify that we want to gather everything but the subject name and time variable:

gather(smiths, key="var", value="value",
       -c(subject,time))
## Error in is.data.frame(x): object 'smiths' not found

Make a column for each subject (= a row for each measurement)

spread(smelt, key=subject, value)
## Error in match(x, table, nomatch = 0L): object 'smelt' not found

Make a column for each value (= a row for each person):

spread(smelt, key=var, value)
## Error in match(x, table, nomatch = 0L): object 'smelt' not found

Take the mean for each variable:

library(dplyr)
smelt %>% group_by(var) %>% summarise(mean=mean(value))
## Error in eval(expr, envir, enclos): object 'smelt' not found

Insist that R calculate a mean:

smelt %>% group_by(var) %>% summarise(mean=mean(value,na.rm=TRUE))
## Error in eval(expr, envir, enclos): object 'smelt' not found

Insist that R calculate a mean, but also report how many values are in each mean:

smelt %>% group_by(var) %>% 
    summarise(mean=mean(value,na.rm=TRUE),
              n=length(na.omit(value)))
## Error in eval(expr, envir, enclos): object 'smelt' not found

If you’re careful you can na.omit() first (be careful, this will remove all rows with any NA values):

smelt %>% na.omit %>% group_by(var) %>% 
    summarise(mean=mean(value,na.rm=TRUE),
              n=n())
